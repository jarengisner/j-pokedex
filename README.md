# j-pokedex

Technologies used:

- HTML
- Javascript
- CSS

API used: 'https://pokeapi.co/api/v2/pokemon/?limit=1000'.

This pokedex was created by connecting to an external API, fetching pokemon,
and details about those pokemon to be displayed on a front end client created with JS.

The pokedex contains a full filter system and a search bar to help make finding pokemon within the project easier.
